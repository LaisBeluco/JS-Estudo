# Tipos de dados

Conforme o EMAScript standard temos 9 tipos de dados:

* Data Types
   * Primmitive / Primitive value
   * Structural 
   * Structural Primitive

## Primitivos

* String
* Number 
* Boolean
* undefined
* Symbol
* BigInt

## Estruturais

* Object
  * Array
  * Map
  * Set
  * Date
  * ...
